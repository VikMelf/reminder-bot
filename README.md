# reminder-bot

**Discord Reminder Bot** — A clean and functional bot for creating reminders, scheduled messages, and task management.


## Features

- Create one-time and recurring reminders
- Schedule messages for a specific date and time
- Task management (add, view, delete reminders)
- Supports natural time formats (e.g. "in 30 minutes", "tomorrow at 15:00")
- Fully supports **English and Ukrainian** languages

## Tech Stack

- **Python**
- **discord.py** (or pycord / nextcord)
- Asyncio for asynchronous operations
